About This Portfolio
----------------------------------------
Each exercise focuses on a different part of front-end development. Together, they show progress in building clean, reusable, and accessible pages.

----------------------------------------
Exercises Included
----------------------------------------
1) Cooking with Andy – Semantic HTML  
2) Machine Gym – CSS Styling  
3) J’s Gallery – Responsive Layout  
4) The Wondering Photographer – Accessibility  

----------------------------------------
How to Use This Repository
----------------------------------------
1. Open each exercise folder and test pages in the browser.  
2. Check links, images, and CSS.  
3. Upload to GitHub.  
4. Copy repo link.  
5. Submit link on Ultra.

Exercise Details
----------------------------------------
Exercise 1 – Cooking with Andy (Semantic HTML)  
- Used semantic elements  
- Structured content clearly

Exercise 2 – Machine Gym (CSS Styling)  
- Used CSS variables

- Responsive layout

Exercise 3 – J’s Gallery (Responsive Layout)  
- Used Grid and Flexbox

Exercise 4 – The Wondering Photographer (Accessibility)
